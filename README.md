# Steven-WabotV2
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg install git
> pkg install ffmpeg
> pkg install imagagick
> pkg install nodejs
> git clone https://github.com/Stevennnm/Steven-WabotV2
> cd Steven-WabotV2
> npm i
```
###### Run
```bash
> node index.js
```
